# OmicsInferenceDeck ---面向多组学癌症分型的全栈AI分析平台（副标题暂定）（多组学癌症亚型识别算法评估平台？）




在传统的生信算法研究中，如果你发明了一个非常创新的算法，想发论文，那么结果分析（还是更应该说是生信分析）是必要的。
这意味着你需要消耗大量的时间、精力计算各种数学指标、绘制论文级别的图表。从零开始写一份这样的代码太麻烦了，因此许多研究者都会去网上找，或者使用AI生成的代码。但不可否认，这样找到的代码质量参差不齐，甚至还是需要消耗大量的时间去调整参数、代码。
我们的OmicsInferenceDeck平台可以解决这一痛点。


主要面向对象是已经研发出了一个算法，得出聚类结果，于是想要评估这个算法，以及使用这个算法发论文的生信研究者。

我们能够提供各种数学指标、论文级别的图表。能够帮研究者省去许多时间，让他们专心于算法的研发和优化


除此之外，研究者在撰写方法学论文时，往往需要耗费大量时间搭建各种对照组来评估算法的优劣。本平台允许用户上传自有数据，灵活配置并运行各种经典算法，直接获取详细的数学评估指标与可视化结果。这大幅缩减了科研人员搭建基线（Baseline）模型的工作量，使他们能够将更多精力集中在自己提出算法的创新与测试上。




在我们这个平台上，如果用户对默认生成的图表不满意，那么可以进入编辑页面，通过我们提供的选项调整。如果还是调整不到满意，那么我们会将绘图的 .R 或 .py 脚本打包，连同数据一起提供给用户下载。用户可以在自己的电脑上运行脚本，完美复现图表并自行微调。
