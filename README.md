# OmicsInferenceDeck ---面向多组学癌症分型的全栈AI分析平台（副标题暂定）

InferenceDeck：面向多组学癌症分型的全栈AI分析平台
项目简介
InferenceDeck（推理控制台）是一个专为计算机、数学及交叉学科研究者打造的全栈AI分析平台，当前主要应用于多组学癌症分型研究。项目名称中的“Inference”代表了严谨的逻辑推理与算法核心，而“Deck”则体现了其作为全栈可视化控制台的强大交互能力。该平台在技术架构上采用了 Vue3 与 FastAPI 的前后端分离模式进行开发。

核心痛点与目标受众
市面上主流的生信分析平台大多面向不会写代码的临床医生或纯生物学者，这类平台通常只提供“黑盒”式的出图功能，缺乏对底层算法参数的调控以及衡量算法好坏的数学指标评估。

与此截然不同，InferenceDeck 的核心受众是懂代码、搞算法的研究者。研究者在撰写方法学论文时，往往需要耗费大量时间搭建各种对照组来评估算法的优劣。本平台允许用户上传自有数据，灵活配置并运行各种经典算法，直接获取详细的数学评估指标与可视化结果。这大幅缩减了科研人员搭建基线（Baseline）模型的工作量，使他们能够将更多精力集中在自己提出算法的创新与测试上。

核心功能特色
严谨且灵活的数据上传与处理：

支持用户上传多种格式的数据（无需强绑定特定的文件后缀名，平台以表格逻辑进行读取），并允许用户灵活指定数据格式（如“第一行为特征名称，第一列为样本名称”或“仅有特征行无表头”等多种组合）。

秉持最高的科研严谨性，平台在数据上传时会进行严格的完整性校验（包括空值检查、纯数字验证、行列名去重等）。对于不合规数据，平台会明确提示错误原因，坚决拒绝任何“自动填充均值”或“自动删行”等会导致实验不可复现的隐性修改。

丰富的多模态/多组学算法库：

后端采用了高度解耦的动态模块映射加载机制，方便算法的迭代与维护。

平台支持用户选择不同的机器学习或多组学聚类算法，目前已集成了包括 K-means、Spectral Clustering（谱聚类）、SNF 以及 NEMO 在内的多种经典算法。

深度的结果评估与可视化展示：

除输出严谨的数学评价指标外，平台还具备强大的可视化分析能力，支持绘制散点图、计算 P值 以及绘制生存曲线等功能。

提供自动化测试支持，包含批量跑参和自动调参功能，全方位赋能算法效果评估。

总结
InferenceDeck 不仅仅是一个低代码 AI 分析平台，它更像是一个专为算法研究者量身定制的“实验对照生成器”。通过规范的数据输入、多维度的算法支持以及科学严谨的报告输出，InferenceDeck 致力于成为计算机方法学与生物信息学交叉领域不可或缺的科研辅助工具。
