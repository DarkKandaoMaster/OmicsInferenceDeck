# OmicsInferenceDeck ---多组学癌症亚型识别算法评估平台


## OmicsInferenceDeck是干什么的？
在传统的生物信息学（多组学癌症亚型识别）算法研究中，当研究者开发出一种创新的聚类算法并准备发表论文时，数学评估与下游生信分析是必不可少的。  
这意味着研究者需要耗费大量的时间和精力：  
1. 计算各种复杂的数学与统计学指标。  
2. 绘制符合高质量期刊要求的“论文级别”图表。  
3. 搭建各种经典算法作为基线对照组来证明新算法的优越性。  

从零开始编写这些评估和可视化代码极其繁琐；在网上寻找开源代码或使用AI生成，质量也往往参差不齐。这极大地分散了研究者的精力。  

OmicsInferenceDeck平台正是为了解决这一痛点而生。（经过市场调研，目前市面上几乎没有解决这一痛点的平台）  
你可以用你写的算法生成一个结果文件，然后根据平台的指示操作，平台便会为你计算指标、绘制图表；  
你也可以上传你自己的输入数据集，选择平台内置经典算法，作为基线对照组计算指标、绘制图表。  


## 目标用户
已经研发出聚类算法、得到初步结果，需要进行下游生信分析，对算法进行全面客观评估与对比，以及撰写学术论文的生物信息学研究者。  


## OmicsInferenceDeck可以提供什么？
### 📊 1. 一键生成“全维评估指标”与“出版级图表”
用户只需上传算法的运行结果，平台即可自动化计算各项复杂的数学评估指标，并生成符合顶级期刊要求的精美图表，包括但不限于：  
* **降维聚类图**（可视化样本分群效果）  
* **差异表达分析图**（火山图等）  
* **GO / KEGG 富集分析图**  
* **生存分析曲线**（Kaplan-Meier 及 P值计算）  
帮您省去手写验证代码的时间，直接获取高质量的分析结果。  

### 🚀 2. 极速搭建基线（Baseline）模型库
在撰写方法学论文时，耗费大量时间搭建对照组是普遍痛点。  
本平台内置了多种经典的多组学/聚类算法（如 `K-means`、`Spectral Clustering (谱聚类)`、`SNF` 以及 `NEMO` 等）。用户可以直接上传自有数据集，灵活配置参数并一键运行这些经典算法，**直接获取详细的对比评估指标**。这大幅缩减了搭建 Baseline 模型的工作量，使横向性能对比变得轻而易举。  

### 🛡️ 3. 严谨且灵活的数据上传与处理机制
秉持最高的科研严谨性，平台在数据处理上绝不含糊：  
* **高灵活度读取**：不强绑定特定的文件后缀名，平台以表格逻辑进行读取。支持用户灵活指定数据格式（如“第一行为特征名称，第一列为样本名称”或“仅有特征行无表头”等）。  
* **零隐性修改**：平台在数据上传时会进行严格的完整性校验（包括空值检查、纯数字验证、行列名去重等）。对于不合规数据，会明确提示错误原因，**坚决拒绝任何“自动填充均值”或“自动删行”等会导致实验不可复现的隐性修改操作**。  

### 🎨 4. “所见即所得”的微调与 100% 脚本级复现（终极自由）
针对科研绘图“众口难调”的特性，我们提供了极具创新性的两段式解决方案：  
* **在线交互编辑**：如果对默认生成的图表细节不满意，可直接进入平台的可视化编辑页面，轻松调整图表的配色、字体、图例及布局。  
* **离线原生复现（杀手锏功能）**：如果 UI 界面的调整依然无法完美满足您的需求，平台支持将当前图表对应的**原生底层绘图脚本（高质量的 `.R` 或 `.py` 代码）连同处理好的数据一键打包下载**。您可以在本地电脑上运行脚本，不仅能 100% 完美复现平台上的图表，还能进行任意维度的代码级微调。  

### ⚙️ 5. 强大的自动化与拓展能力
* **自动化测试支持**：包含批量跑参和自动调参功能，全方位赋能算法效果评估。  
* **高度解耦的算法库**：后端采用了高度解耦的动态模块映射加载机制，极其方便后续新算法的接入、迭代与维护。
