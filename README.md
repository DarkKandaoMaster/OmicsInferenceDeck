# OmicsInferenceDeck ---多组学癌症亚型识别算法评估平台


## OmicsInferenceDeck是干什么的？
在传统的生物信息学（多组学癌症亚型识别）算法研究中，当研究者开发出一种创新的聚类算法并准备发表论文时，数学评估与下游生信分析是必不可少的。  
这意味着研究者需要耗费大量的时间和精力：  
1. 计算各种复杂的数学与统计学指标。
2. 绘制符合高质量期刊要求的“论文级别”图表。
3. 搭建各种经典算法作为基线对照组来证明新算法的优越性。
从零开始编写这些评估和可视化代码极其繁琐；在网上寻找开源代码或使用AI生成，质量也往往参差不齐。这极大地分散了研究者的精力。

OmicsInferenceDeck平台正是为了解决这一痛点而生。（经过市场调研，目前市面上完全没有解决这一痛点的平台）

核心功能特色
严谨且灵活的数据上传与处理：

支持用户上传多种格式的数据（无需强绑定特定的文件后缀名，平台以表格逻辑进行读取），并允许用户灵活指定数据格式（如“第一行为特征名称，第一列为样本名称”或“仅有特征行无表头”等多种组合）。

秉持最高的科研严谨性，平台在数据上传时会进行严格的完整性校验（包括空值检查、纯数字验证、行列名去重等）。对于不合规数据，平台会明确提示错误原因，坚决拒绝任何“自动填充均值”或“自动删行”等会导致实验不可复现的隐性修改。

丰富的多模态/多组学算法库：

后端采用了高度解耦的动态模块映射加载机制，方便算法的迭代与维护。

平台支持用户选择不同的机器学习或多组学聚类算法，目前已集成了包括 K-means、Spectral Clustering（谱聚类）、SNF 以及 NEMO 在内的多种经典算法。

深度的结果评估与可视化展示：

除输出严谨的数学评价指标外，平台还具备强大的可视化分析能力，支持绘制散点图、计算 P值 以及绘制生存曲线等功能。

提供自动化测试支持，包含批量跑参和自动调参功能，全方位赋能算法效果评估。



如果用户对平台默认生成的图表细节不满意，可直接进入交互式编辑页面。通过平台提供的可视化配置选项，轻松调整图表的配色、字体、图例及布局。

1. 一键生成“全维评估指标”与“出版级图表”
