# 环境配置 (Python + R)

OmicsInferenceDeck 的后端是 FastAPI（Python），其中多组学聚类、差异表达、富集分析、
各类统计指标与绘图通过 `Rscript` 调用 R 脚本完成。因此**必须同时配置好 Python 和 R 两套环境**，
并保证 `Rscript` 可在命令行（PATH）中直接调用。

> 目标平台：Windows 10/11。下文命令以 PowerShell 为准。

---

## 1. Python 环境

后端依赖见 [requirements.txt](requirements.txt)。建议使用 Conda 虚拟环境（推荐 Python 3.11）。

### 方式 A：使用已有 Conda 环境

本机已有环境 `myenv`：

```powershell
& "D:\Programs\DevEnvironments\Anaconda\anaconda3\envs\myenv\python.exe" -m pip install -r requirements.txt
```

### 方式 B：新建 Conda 环境

```powershell
conda create -n omics python=3.11 -y
conda activate omics
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

### 关键依赖说明

| 类别 | 包 |
| --- | --- |
| Web 服务 | fastapi, uvicorn, pydantic, python-multipart |
| 数据处理 | pandas, numpy, scipy, **pyarrow**（与 R 的 `arrow` 包共同完成 parquet 数据交换）, openpyxl |
| 机器学习/聚类 | scikit-learn, umap-learn, snfpy, pyrea |
| 生存/富集 | lifelines, gseapy |
| 绘图 | matplotlib, seaborn, adjustText, PyMuPDF, Pillow, svgutils |

### 启动后端

```powershell
conda activate omics   # 或使用 myenv
uvicorn backend.main:app --reload
```

---

## 2. R 环境

### 2.1 安装 R 与 Rtools

1. 安装 **R ≥ 4.3**：<https://cloud.r-project.org/bin/windows/base/>
2. 安装 **Rtools**（编译源码包必需）：<https://cloud.r-project.org/bin/windows/Rtools/>
3. 确认 `Rscript` 在 PATH 中（后端通过 `subprocess.run(["Rscript", ...])` 调用）：

   ```powershell
   Rscript --version
   ```

   若提示找不到命令，将 R 安装目录下的 `bin`（如 `C:\Program Files\R\R-4.4.1\bin`）加入系统环境变量 PATH。

### 2.2 一键安装 R 依赖

```powershell
Rscript backend/install_r_packages.R
```

该脚本会按需安装下列依赖（已安装的会自动跳过）。

### 2.3 R 依赖清单

**CRAN**

| 包 | 用途 |
| --- | --- |
| arrow | 读写 parquet（与 Python `pyarrow` 对接，所有 R 脚本的数据入口） |
| ggplot2, ggrepel | 火山图、富集气泡/柱状图 |
| pheatmap | 差异表达热图 |
| svglite | SVG 输出 |
| dplyr, stringr | 数据整理 |
| cluster, clusterCrit | 聚类内部指标（轮廓系数等） |
| survival | 生存分析 / 临床指标 |
| SNFtool | SNF / NEMO 谱聚类 |

**Bioconductor**

| 包 | 用途 |
| --- | --- |
| limma, edgeR | 差异表达分析 |
| clusterProfiler, org.Hs.eg.db, AnnotationDbi | GO / KEGG 富集分析 |

**算法包（源码安装，需 Rtools）**

| 包 | 用途 |
| --- | --- |
| NEMO | NEMO 聚类算法 |
| PintMF | PintMF 聚类算法 |
| MOSD | MOSD 聚类算法 |

> 以上算法包由 `install_r_packages.R` 自动安装，无需手动操作。

## 3. 验证

```powershell
# Python
& "D:\Programs\DevEnvironments\Anaconda\anaconda3\envs\myenv\python.exe" -c "import fastapi, pandas, sklearn, umap, gseapy; print('Python OK')"

# R（检查关键包能否加载）
Rscript -e "lapply(c('arrow','limma','edgeR','clusterProfiler','SNFtool','ggplot2'), requireNamespace); cat('R OK\n')"
```

两条命令均无报错即配置完成。
